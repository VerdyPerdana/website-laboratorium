  • laboratorium Praktikum PTI

    Kegiatan Praktikum di Laboratorium PTI ini merupakan salah satu langkah strategis untuk meningkatkan kompetensi kita dalam bidang teknologi informasi. Laboratorium PTI tidak hanya menjadi tempat belajar, tetapi juga ruang eksplorasi, inovasi, dan pengembangan keterampilan teknologi. Berikut beberapa Praktium nya :

    • Membuat Website
    • Merancang Topologi Jaringan
    • Desain Menggunakan Canva
    • Menghitung Data Excel
    • Membuat Database
  • Kompetisi Prodi PTI

    Kegiatan lomba ini bukan sekadar ajang kompetisi untuk Prodi Pendidikan Teknologi Informasi (PTI), tetapi juga wadah untuk mengasah kemampuan, berbagi pengetahuan, dan memperkuat solidaritas di antara kita sebagai insan yang terus belajar dan berkembang dalam bidang teknologi informasi. Berikut Daftar Kompetisi yang akan datang :

    • Networking
    • Desain Poster Canva
    • Website Html dan Css
    • Aplikasi Mobile
  • Project Prodi PTI

    Setiap naik semester pasti ada project UAS, dan untuk mendapatkan nilai yang bagus anak Praktikum harus mengerjakan nya dengan baik dan sempurna, bukan cuman sekedar memenuhi tugas tetapi untuk mendapatkan ilmu dan pengalaman yang akan digunakan untuk masa depan nanti. Berikut beberapa Project UAS Program Studi PTI :

    • Project Website
    • Project Pendidikan E-Learning
    • Project Jaringan Komputer
    • Project IoT (Internet of Things)
    • Project Aplikasi Mobile
